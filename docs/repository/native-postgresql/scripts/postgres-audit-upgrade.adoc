[[change-1]]
=== Change 1

Since version: 4.4.1

        Updates the audit monthly partition creation procedure to support creating partitions in the past using a negative argument.

[cols="1,1,3", options="header"]
|===
| Object | Change type | Description
| `audit_create_monthly_partitions` | Modified procedure | Supports creating audit partitions in past months using a negative argument.
| `ma_audit_event_<month>` | New generated partition | Creates monthly audit event partitions.
| `ma_audit_delta_<month>` | New generated partition | Creates monthly audit delta partitions.
| `ma_audit_ref_<month>` | New generated partition | Creates monthly audit reference partitions.
| `ma_audit_delta_<month>_fk` | New generated foreign key | Links monthly audit delta partitions to matching audit event partitions.
| `ma_audit_ref_<month>_fk` | New generated foreign key | Links monthly audit reference partitions to matching audit event partitions.
|===
