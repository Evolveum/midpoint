Adds role analysis cluster and role analysis session tables, triggers, and indexes.

[cols="1,1,3", options="header"]
|===
| Object
| Change type
| Description

| `m_role_analysis_cluster`
| New table
| Stores role analysis cluster objects.

| `m_role_analysis_cluster_oid_insert_tr`
| New trigger
| Reserves OID rows for inserted role analysis clusters.

| `m_role_analysis_cluster_update_tr`
| New trigger
| Maintains update metadata for role analysis clusters.

| `m_role_analysis_cluster_oid_delete_tr`
| New trigger
| Releases OID rows for deleted role analysis clusters.

| New index
| Supports lookup by parent reference target OID.

| `m_role_analysis_cluster_parentRefTargetType_idx`
| New index
| Supports lookup by parent reference target type.

| `m_role_analysis_cluster_parentRefRelationId_idx`
| New index
| Supports lookup by parent reference relation.

| `m_role_analysis_session`
| New table
| Stores role analysis session objects.

| `m_role_analysis_session_oid_insert_tr`
| New trigger
| Reserves OID rows for inserted role analysis sessions.

| `m_role_analysis_session_update_tr`
| New trigger
| Maintains update metadata for role analysis sessions.

| `m_role_analysis_session_oid_delete_tr`
| New trigger
| Releases OID rows for deleted role analysis sessions.
