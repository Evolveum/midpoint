[[change-13]]
== Change 13

Since version: 4.7

Adds simulation result, processed object, mark, and effective mark tables, triggers, functions, and indexes.

[cols="1,1,3", options="header"]
|===
| Object
| Change type
| Description

| `m_simulation_result`
| New table
| Stores simulation result objects.

| `m_simulation_result_oid_insert_tr`
| New trigger
| Reserves OID rows for inserted simulation results.

| `m_simulation_result_update_tr`
| New trigger
| Maintains update metadata for simulation results.

| `m_simulation_result_oid_delete_tr`
| New trigger
| Releases OID rows for deleted simulation results.

| `ObjectProcessingStateType`
| New enum type
| Stores processed object states for simulation results.

| `m_simulation_result_processed_object`
| New table
| Stores processed objects belonging to simulation results.

| `m_simulation_result_processed_object_default`
| New partition
| Default partition for processed simulation objects.

| `m_simulation_result_create_partition`
| New function
| Creates processed-object partitions for partitioned simulation results.

| `m_simulation_result_create_partition`
| New trigger
| Creates processed-object partition when a partitioned simulation result is inserted.

| `m_simulation_result_delete_partition`
| New function
| Drops processed-object partitions when a simulation result is deleted.

| `m_simulation_result_delete_partition`
| New trigger
| Deletes processed-object partition before deleting a simulation result.

| `m_processed_object_event_mark`
| New table
| Stores event mark references for processed simulation objects.

| `m_processed_object_event_mark_default`
| New partition
| Default partition for processed-object event mark references.

| `m_mark`
| New table
| Stores mark objects.

| `m_ref_object_effective_mark`
| New table
| Stores effective mark references for objects.

| `m_ref_object_effective_mark_targetOidRelationId_idx`
| New index
| Supports reverse lookup of effective mark references.

|===
