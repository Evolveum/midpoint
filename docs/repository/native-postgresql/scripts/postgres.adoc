= Native PostgreSQL Schema Script: postgres.sql
:page-toc: top

xref:../schema.adoc[Back to native PostgreSQL schema overview]

This page documents schema objects parsed from `config/sql/native/postgres.sql`.
The detailed documentation is split into logical region pages because this schema script is large.

== Region documentation

* xref:postgres/infrastructure.adoc[Infrastructure] - Schema, extensions, enum types, shared identifiers, repository metadata, and support routines.
* xref:postgres/references.adoc[References] - Tables storing object and container references such as archetype, projection, role membership, and approver references.
* xref:postgres/focus-and-users.adoc[Focus and users] - Focus base data, users, generic objects, focus identities, and focus-related references.
* xref:postgres/roles-and-services.adoc[Roles and services] - Abstract roles, roles, services, archetypes, policies, applications, and related object tables.
* xref:postgres/organization.adoc[Organization] - Organization objects, parent organization references, and organization hierarchy closure support.
* xref:postgres/resources-and-shadows.adoc[Resources and shadows] - Resources, shadows, shadow partitions, connectors, and resource object reference data.
* xref:postgres/system-objects.adoc[System objects] - Node, system configuration, security policy, and other system-level repository objects.
* xref:postgres/reports-and-dashboards.adoc[Reports and dashboards] - Reports, report data, dashboards, object collections, forms, function libraries, and related UI/reporting objects.
* xref:postgres/role-analysis.adoc[Role analysis] - Role analysis sessions, clusters, outliers, detected patterns, and related containers.
* xref:postgres/lookup-tables.adoc[Lookup tables] - Lookup table objects and their key-value row data.
* xref:postgres/cases-and-certification.adoc[Cases and certification] - Case management, access certification campaigns, certification cases, work items, and reviewer references.
* xref:postgres/templates-and-messages.adoc[Templates and messages] - Object templates, include references, sequences, forms, function libraries, and message templates.
* xref:postgres/assignments.adoc[Assignments] - Assignment and inducement containers, assignment metadata, and assignment reference tables.
* xref:postgres/tasks.adoc[Tasks] - Task objects, task containers, affected objects, triggers, and operation execution data.
* xref:postgres/simulations.adoc[Simulations] - Simulation result objects, processed simulation objects, and simulation event mark references.
* xref:postgres/marks.adoc[Marks] - Mark objects used to classify and annotate repository objects and events.
* xref:postgres/schema-objects.adoc[Schema objects] - Schema repository objects.
* xref:postgres/extension-items.adoc[Extension items] - Extension item catalog and indexed extension value storage support.
* xref:postgres/schema-versioning.adoc[Schema versioning] - Procedures and metadata used to track and apply native repository schema changes.

| xref:postgres/organization.adoc[Organization]
