= Upgrade installation

The `upgrade-installation` command copies files from the new midPoint distribution to the existing midPoint installation directory, replacing files that already exist.
Specify the location of the new midPoint distribution using the `--distribution-directory <PATH>` option.

By default, the installation directory is determined as the parent directory of the midPoint home directory.
To use a different installation directory, specify it using the `--installation-directory <PATH>` option.

Before replacing existing files, Ninja can back up the files that would be overwritten. To enable this backup, use the `--backup-installation-directory` option.
The backup is created automatically in the installation directory as `.backup-<timestamp>`.
